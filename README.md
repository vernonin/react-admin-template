## 简介
**由`react`+`antd`开发的简洁后台管理系统模板**
- charts图标
- ant-design组件
- markdown
- ......


> `react@17`、`react-router-dom@6`、`antd`、`ant-design/charts`、`styled-components`


## 下载
```
git clone https://github.com/vernonin/react-admin-template.git
```

## 安装依赖
```
yarn install
```
## 运行与发布
```
yarn start && yarn bulid
```
